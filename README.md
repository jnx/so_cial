# so_cial (So Social)

This gem adds atm. twitters sharebutton on your page

## Installation

Add this line to your application's Gemfile:

    gem 'so_cial'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install so_cial

## Usage

In your viewfile add <%= social_button_for(:twitter) %> to get the twitter sharebutton
